# Frontend Configuration

## Next.js Configuration

The main Next.js configuration is in `next.config.js`:

```javascript
/** @type {import('next').NextConfig} */
const nextConfig = {
  // Environment variables exposed to the browser
  env: {
    NEXT_PUBLIC_API_URL: process.env.NEXT_PUBLIC_API_URL,
  },

  // Image optimization
  images: {
    domains: ['your-cdn-domain.com'],
  },

  // Rewrites for API proxy (optional)
  async rewrites() {
    return [
      {
        source: '/api/:path*',
        destination: `${process.env.NEXT_PUBLIC_API_URL}/:path*`,
      },
    ];
  },
};
```

## Environment Variables

### Required Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `NEXT_PUBLIC_API_URL` | Backend API base URL | `http://localhost:8000/api/v1` |

### Optional Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `NEXT_PUBLIC_SENTRY_DSN` | Sentry error tracking DSN | `https://...@sentry.io/...` |
| `NEXT_PUBLIC_ENABLE_ANALYTICS` | Enable analytics tracking | `true` |

## API Client Configuration

The API client in `src/lib/api.ts` can be customized:

```typescript
const apiClient = axios.create({
  baseURL: process.env.NEXT_PUBLIC_API_URL,
  timeout: 30000,
  headers: {
    'Content-Type': 'application/json',
  },
});

// Add auth token to requests
apiClient.interceptors.request.use((config) => {
  const token = getToken();
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});
```

## Authentication Configuration

Configure authentication in `src/lib/auth.ts`:

```typescript
export const authConfig = {
  // Session duration in seconds
  sessionMaxAge: 60 * 60 * 24 * 7, // 1 week

  // Redirect paths
  loginPath: '/login',
  dashboardPath: '/dashboard',

  // Protected routes
  protectedRoutes: ['/dashboard', '/reports', '/admin'],
};
```

## Build Configuration

### Production Build

```bash
npm run build
```

### Build Output

The build output is in `.next/` directory. For deployment:

```bash
npm run start
```

### Static Export (Optional)

For static hosting, configure in `next.config.js`:

```javascript
module.exports = {
  output: 'export',
};
```
