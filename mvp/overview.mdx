# Gistly Call Analytics - MVP Overview

Gistly Call Analytics is a comprehensive meeting intelligence platform that transforms how organizations analyze and act on customer conversations.

## What is Gistly?

Gistly is an AI-powered platform that:
- **Records and transcribes** customer meetings across multiple channels
- **Analyzes conversations** using advanced AI to extract insights, sentiment, and action items
- **Generates reports** including red flag analysis, meeting summaries, and trend data
- **Integrates** with existing CRM systems (Attio, custom integrations)
- **Processes leads** through intelligent automation workflows

## MVP Scope

The Minimum Viable Product includes:

### Core Features
- Meeting recording and transcription
- AI-powered conversation analysis
- Red flag detection and scoring
- Report generation (meetings, trends, red flags)
- Lead processing and CRM integration
- Organization and user management
- Dashboard and analytics

### Technical Components
- **Frontend**: Next.js dashboard with shadcn/ui components
- **Backend**: Django REST API with PostgreSQL database
- **AI Processing**: OpenAI GPT integration for analysis
- **Deployment**: AWS infrastructure with CI/CD pipelines

## Target Users

1. **Sales Teams** - Analyze customer calls for coaching and insights
2. **Customer Success** - Monitor health signals and identify at-risk accounts
3. **Product Teams** - Gather feedback and identify feature requests
4. **Management** - Track team performance and key metrics

## Key Metrics Tracked

- **Meeting Duration** - Length of customer conversations
- **Red Flags** - Potential issues or concerns raised by customers
- **Speaker Ratios** - Talk-time balance between participants
- **Sentiment Trends** - Customer sentiment over time
- **Action Items** - Tasks and follow-ups identified in meetings
