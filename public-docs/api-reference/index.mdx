# API Reference

Explore the Gistly API to integrate conversation intelligence into your applications.

## Authentication

All API requests require authentication using a Bearer token:

```http
Authorization: Bearer YOUR_API_TOKEN
```

## Base URL

- **Production**: `https://api.gistly.ai`

## Rate Limits

API requests are rate limited to 1000 requests per hour per API key.
