## Authentication

All API requests require authentication using a Bearer token:

```http
Authorization: Bearer YOUR_API_KEY
```

## Base URL

- **Production**: `https://api.gistly.com`

## Rate Limits

API requests are rate limited to 1000 requests per hour per API key.
