# Gistly API

The Gistly API lets you programmatically access conversation intelligence data — meetings, transcripts, analytics, reports, and more.

## Base URL

```
https://api.gistly.com
```

## Authentication

All API requests require a Bearer token in the `Authorization` header:

```http
Authorization: Bearer YOUR_API_TOKEN
```

To get your API token, log in to [console.gistly.ai](https://console.gistly.ai) and navigate to **Settings > API Keys**.

See the [Authentication guide](/api-reference/authentication) for details on obtaining and managing tokens.

## Rate Limits

API requests are rate limited to **1000 requests per hour** per API key. Rate limit headers are included in every response:

```http
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1704110400
```

## Response Format

All responses use JSON. Successful responses include a `message` field:

```json
{
  "message": "Meetings fetched successfully.",
  "meetings": [...],
  "pagination": {
    "page": 1,
    "page_size": 10,
    "total_count": 42,
    "total_pages": 5
  }
}
```

Error responses follow a consistent format. See [Error Handling](/api-reference/error-handling) for details.
